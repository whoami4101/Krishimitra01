# KrishiMitra - Smart Farming App

## Disease Detection Setup

### 1. Start Backend
```bash
./START_BACKEND.sh
```

### 2. Start App
```bash
npm start
```

### 3. Use Disease Detection
- Go to **Farmer Input** tab
- Take/upload plant leaf photo
- Get AI disease detection results

## Supported Crops
Apple, Blueberry, Cherry, Corn, Grape, Orange, Peach, Pepper, Potato, Raspberry, Soybean, Squash, Strawberry, Tomato

## Features
- Real-time sensor monitoring
- AI disease detection (38 diseases)
- Weather forecasting
- Historical data charts
- Satellite imagery
- Multi-language support
